
# 中風患者復健效果分析：Python 實作與初步結果解讀

## 1. 分析目的

本分析目的為評估中風患者在復健過程中的功能恢復效果。由於資料沒有明確的事件發生時間與 censoring 結構，因此不建議使用 Cox regression 作為主模型。本次採用線性混合效果模型（Linear Mixed Effects Model, LMM）作為 GEE 與 Cox 之外的主要分析方法。

## 2. 資料結構

使用工作表：`Data`

樣本數：101 位個案。

主要 outcome：ADL total score

時間點：

- pre：復健前
- P：中間或第一次追蹤
- PP：後續追蹤或復健後

ID 規則：

- 1XX：缺血性中風
- 2XX：出血性中風

## 3. 描述性結果：ADL 隨時間變化

初步上可以看到 ADL 平均值由 pre 的 39.65 上升到 P 的 71.33，再上升到 PP 的 80.05。

|           |   n |   mean_change |   sd_change |        t |   p |

主要結果：

- pre 到 P：平均改善 31.53 分，p = <0.001

- pre 到 PP：平均改善 38.49 分，p = <0.001

這表示 ADL 在追蹤過程中有明顯改善，符合「可以做恢復效果分析」的資料條件。

## 5. 主模型：線性混合效果模型

### 模型 A：主模型

模型：

```text
ADL ~ time_num + ageCE365 + sex + CVA_type + T_hour + (1 | ID)
```

其中 `(1 | ID)` 表示每位病人有自己的 baseline 差異，也就是 random intercept。

模型 A 主要係數：

|                  |   Coef. |   Std.Err. | z      | P>|z|   | [0.025   | 0.975]   |
|:-----------------|--------:|-----------:|:-------|:--------|:---------|:---------|
| Intercept        | 169.966 |     22.422 | 7.580  | 0.000   | 126.018  | 213.913  |
| C(sex)[T.2]      |  -4.372 |      3.792 | -1.153 | 0.249   | -11.803  | 3.060    |
| C(CVA_type)[T.2] |  -0.171 |      4.42  | -0.039 | 0.969   | -8.834   | 8.491    |
| time_num         |  19.67  |      1.07  | 18.382 | 0.000   | 17.573   | 21.768   |
| ageCE365         |  -0.7   |      0.14  | -4.991 | 0.000   | -0.975   | -0.425   |
| T_hour           | -13.882 |      3.375 | -4.113 | 0.000   | -20.497  | -7.267   |
| Group Var        | 275.763 |      4.177 |        |         |          |          |

解讀重點：

- `time_num` 係數為 19.67，代表每往後一個時間點，ADL 平均增加約 19.67 分，且 p 值高度顯著。
- 年齡係數為 -0.70，代表年齡越高，ADL 整體分數傾向較低。
- `T_hour` 在此模型中呈負向，這不應直接解釋為「復健時數有害」。較合理的解釋是嚴重病人可能被安排更多治療，因此存在 indication bias 或 confounding by severity。

### 模型 B：加入 NIHSS 與交互作用

模型：

```text
ADL ~ time_num*T_hour + time_num*NIHSS + ageCE365 + sex + CVA_type + (1 | ID)
```

模型 B 主要係數：

|                  |   Coef. |   Std.Err. | z      | P>|z|   | [0.025   | 0.975]   |
|:-----------------|--------:|-----------:|:-------|:--------|:---------|:---------|
| Intercept        | 144.239 |     29.245 | 4.932  | 0.000   | 86.920   | 201.558  |
| C(sex)[T.2]      |  -5.962 |      4.392 | -1.357 | 0.175   | -14.570  | 2.647    |
| C(CVA_type)[T.2] |  -4.557 |      6.639 | -0.686 | 0.492   | -17.570  | 8.455    |
| time_num         |  32.005 |     12.493 | 2.562  | 0.010   | 7.520    | 56.490   |
| T_hour           |  -7.776 |      4.683 | -1.661 | 0.097   | -16.953  | 1.402    |
| time_num:T_hour  |  -3.018 |      2.307 | -1.308 | 0.191   | -7.539   | 1.504    |
| NIHSS            |  -0.807 |      0.33  | -2.447 | 0.014   | -1.454   | -0.161   |
| time_num:NIHSS   |   0.468 |      0.161 | 2.911  | 0.004   | 0.153    | 0.783    |
| ageCE365         |  -0.707 |      0.16  | -4.413 | 0.000   | -1.021   | -0.393   |
| Group Var        | 270.878 |      4.67  |        |         |          |          |

解讀重點：

- `time_num` 仍然顯著，表示整體恢復趨勢存在。
- `NIHSS` baseline effect 為負向，表示較嚴重中風患者的 ADL 起點較低。
- `time_num:NIHSS` 為正向且顯著，可能表示 NIHSS 較高者雖然起點較低，但在後續時間點有較大的改善空間。這需要搭配圖形與臨床解讀，不能直接理解成嚴重度高一定恢復更好。
- `time_num:T_hour` 在目前模型中未達顯著，表示在控制其他變數後，資料尚不足以支持「復健時數改變 ADL 恢復斜率」的明確結論。

## 7. 結論

這份資料可以做恢復效果分析，而且比 Cox regression 更適合使用 longitudinal framework。

建議主分析模型：

```text
Linear Mixed Effects Model
```

核心研究問題可寫成：

> 中風患者在復健追蹤期間是否呈現顯著功能恢復？年齡、中風嚴重度、復健時數與中風類型是否影響其 ADL 恢復軌跡？

目前初步分析支持：

1. ADL 隨時間明顯改善。
2. 年齡較高者 ADL 整體分數較低。
3. NIHSS 與恢復軌跡有關，但需要避免過度因果解讀。
4. 復健時數的係數受到病情嚴重度混雜影響，不宜直接解釋為治療效果。
